# Microsoft Azure Terraform Introduction Workshop
This repository contains all of the content required to deliver an on-premises instructor-led introductory Terraform training course. All of the Labs will demonstrate through interactive hands-on practice how to use Terraform to manage cloud infrastructure in Microsoft Azure.
